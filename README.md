# AQI Cloud Monitoring PoC

Proyecto final basado en un sistema de monitoreo de calidad del aire utilizando tecnologías cloud y contenedores.

## Tecnologías utilizadas

- Flask
- PostgreSQL
- Docker
- Docker Compose
- GitHub Actions
- Docker Hub
- Render
